## Prerequisites

- Python 3.9+
- A Sector8 API key ([book a demo](https://sector8.ai/contact) to get one)

## 1. Install the SDK

```bash
pip install sector8-sdk
```

## 2. Set your API key

```bash
export SECTOR8_API_KEY=sk-your-key-here
```

## 3. Wrap your first tool call

```python
from sector8 import GuardClient

guard = GuardClient(api_key="sk-your-key-here")

# Evaluate before executing
result = guard.evaluate(
    tool_name="bash",
    arguments={"command": "ls /tmp"}
)

if result.decision == "ALLOW":
    print("Allowed:", result.decision_trace_id)
else:
    print("Blocked:", result.reason_code)
    print("Trace ID:", result.decision_trace_id)
```

## 4. Verify it works

A safe command returns `ALLOW`:

```bash
python your_script.py
# Allowed: bd970b1f-5336-41ef-a63a-ebbf8e4db839
```

A destructive command is blocked:

```python
result = guard.evaluate(
    tool_name="bash",
    arguments={"command": "rm -rf /"}
)
# decision: DENY
# reason_code: MCP_SHELL_INJECTION_BLOCKED
# has_forensic_payload: true
```

<Check>
  Both calls routed through the Guard Module with tamper-evident audit trails.
</Check>
